# 🪪 OpenGradient x402 Card Generator

> Generate your verifiable, on-chain user card for the OpenGradient ecosystem.

[🌐 Live App](https://open-gradient-card.vercel.app/)

---

## ✨ Features

- 🖼️ **Custom Avatar**: Upload or select your profile image
- 🎖️ **Rank Badges**: Display your community tier:
  `Quantling` → `Quant` → `Senior Quant` → `Master` → `Elite` → `The Supreme Quant` → `OG` / `ALPHA OG`
- 🎨 **Auto-Generated Card**: Clean, branded design with gradient styling
- 📥 **One-Click Download**: Save your card as PNG for sharing
- 📅 **Date Selector**: Add join date or milestone to your card

---

## 🚀 How to Use

1. Visit [open-gradient-card.vercel.app](https://open-gradient-card.vercel.app/)
2. Upload your avatar or choose a default
3. Select your rank/badge
4. (Optional) Pick a significant date
5. Click **Generate Card** → **Download**
6. Share on X with `#OpenGradient #x402` to participate in community rewards [[5]]
